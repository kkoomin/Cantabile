# Cantabile

iPad application for music students to read music sheets, manage daily schedules and practice hours

## Feature

1. Calendar
2. PDF score reader
3. Practice log
   - Timer
   - Daily ToDo List
4. Setting
   - Metronome
   - Recorder
   - Personal setting
