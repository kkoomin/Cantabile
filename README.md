# Cantabile

iPad application for music students to read music sheets, manage daily schedules and practice hours

## Feature

1. Calendar
2. PDF score reader
3. Practice log
   - Timer
   - Daily ToDo List
4. Setting
   - Metronome
   - Recorder
   - Personal setting

## Project Schedule

- MVP features

  - calendar schedule add form

  - score list rendering
  - Metronome
  - timer
  - daily to do list
  - Recorder

- Final version features
  - MVP features
  - Recorder
  - App lock screen
  - Clear app data functionality
